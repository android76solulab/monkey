# monkey
A select media file library for android.

## Download

Gradle:
```groovy
repositories {
    jcenter()
}

dependencies {
    compile 'com.miraclehen:monkey:1.0.0'
}
```

Maven:
```groovy
<dependency>
  <groupId>com.miraclehen</groupId>
  <artifactId>monkey</artifactId>
  <version>1.0.0</version>
  <type>pom</type>
</dependency>
```
